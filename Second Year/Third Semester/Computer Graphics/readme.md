# Basics of Computer Graphics

## Overview

Big course that consist of different topics and every topic can be its own different course. A lot of similarities from linear algebra and geometry.

## Grade evaluation:
The course has a lot of tasks and has 2 theoretical tests, consisting of questions from the presentations the professor gives.
The 2 tests are both 25 points. You can pass very easily. You can get points from:

1. Theoretical tests   - 25 points x 2
2. Assistant tasks     - 2 points x 5
3. Homework            - 5 points x 2
4. Bonuses (either in assistant tasks/homework or during lections)
5. Project
