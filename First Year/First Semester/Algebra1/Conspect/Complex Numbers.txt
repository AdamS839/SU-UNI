3.Поле
3.1.Определение на поле
	Свойства на полето:
	1. (a + b) + c = a + (b + c) - асоциативност
	2. a + b = b + a - комутативнст
	3. съществува нулев елемент 0, за който: a + 0 = a 
	4. за всеки елемент a ∈ M, съществува b ∈ M, за който: a + b = b + a = 0 
	5. (a.b).c = a.(b.c) - асоциативност
	6. a.b = b.a - комутативност
	7. (a + b).c = a.c + b.c - дистрибутивност
	8. съществува единичен елемент 1 =/= 0, за който: 1.a = a, за всяко a ∈ M
	9. за всеки ненулев елемент 0 =/= a ∈ M, за който: a.c = c.a = 1
